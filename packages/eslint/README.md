# @yopem/eslint-config

Yopem ESLint configuration.

## Installation

```sh
npm install @yopem/eslint-config
# or
pnpm add @yopem/eslint-config
# or
yarn add @yopem/eslint-config
# or
bun add @yopem/eslint-config
# or
deno install npm:@yopem/eslint-config
```
