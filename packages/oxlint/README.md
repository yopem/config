# @yopem/oxlint-config

Yopem oxlint configuration.

## Installation

```sh
npm install @yopem/oxlint-config
# or
pnpm add @yopem/oxlint-config
# or
yarn add @yopem/oxlint-config
# or
bun add @yopem/oxlint-config
# or
deno install npm:@yopem/oxlint-config
```
