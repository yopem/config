# @yopem/oxfmt-config

Yopem oxfmt configuration.

## Installation

```sh
npm install @yopem/oxfmt-config
# or
pnpm add @yopem/oxfmt-config
# or
yarn add @yopem/oxfmt-config
# or
bun add @yopem/oxfmt-config
# or
deno install npm:@yopem/oxfmt-config
```
